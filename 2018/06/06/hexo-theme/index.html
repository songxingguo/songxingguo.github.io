Hexo主题修改

发表于 2018-06-06 | 更新于 2018-10-13 | 分类于 博客搭建 | 评论数： | 阅读次数：

  • 前提

      已经使用Hexo搭建一个博客

  • 选取自己喜欢的主题

      在Hexo 主题列表 中选取一个自己喜欢的主题，

      我这儿选取的是 hiker 主题

  • 将主题clone到themes目录下

    • 进入博客根目录，将代码clone到 themes/hiker 目录下

      效果：

    • 然后进入主题文件目录下，执行 npm install 安装主题依赖的资源

      效果：

  • 修改博客_config.yml文件

    • 打开博客的—config.yml文件，找到 theme 字段并修改为 hiker

  • 重新启动服务

    • 进入博客根目录，重新启动服务

    • 点击 http://localhost:4000 查看
