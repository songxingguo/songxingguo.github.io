前端相关

this

this 对象是在运行时 基于函数的执行环境绑定的 ；在 全局函数 中，this 等于 window ，而当 函数被作为某个对象的方法调用 时，this 等于那个对象 。不过，匿名函数的执行环境具有全局性 ，因此其 this 对象通常指向 window 。
